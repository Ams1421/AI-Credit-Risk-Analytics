# AI Credit Risk Analytics Dashboard

---

# 1. Introduction

## Background

Financial institutions receive a large number of loan applications every day. Evaluating each application manually is time-consuming and increases the possibility of inconsistent decision-making.

Modern banking organizations increasingly rely on Business Intelligence and Machine Learning to automate credit risk assessment, improve operational efficiency, and reduce financial losses.

This project demonstrates a complete end-to-end analytics solution that integrates data engineering, SQL analytics, interactive dashboards, and predictive modeling into a unified workflow.

---

# 2. Project Objectives

The primary objectives of this project are:

- Analyze historical loan application data.
- Understand factors influencing loan approval.
- Identify high-risk applicants.
- Develop interactive business dashboards.
- Build a machine learning model to predict loan approval.
- Provide actionable insights for financial institutions.

---

# 3. Project Scope

The project covers the complete analytics lifecycle:

- Data Collection

- Exploratory Data Analysis

- Business Reporting

The project is designed as a portfolio demonstrating practical Data Analytics and Machine Learning skills.

# 5. Dataset Description

The dataset contains historical loan application records used for classification.

### Input Features

- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Term
- Credit History
- Property Area

### Target Variable

Loan Status

- Approved (Y)
- Rejected (N)
